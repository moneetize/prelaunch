# Moneetize Prelaunch

Starter repository for the Moneetize prelaunch project.

## Status

This repository has been initialized with a lightweight scaffold so the project can be connected locally and on GitHub without committing to a specific framework too early.

## Included

- `README.md` for project orientation
- `.gitignore` with common web and local development exclusions
- `.editorconfig` for consistent formatting defaults
- `.gitattributes` for normalized text handling across platforms
